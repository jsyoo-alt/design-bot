# 카카오 비즈보드 소재봇 — 사용 가이드

> 슬랙에서 카카오 비즈보드 광고 소재(1029×258px)를 자동 생성하는 봇입니다.  
> 개발 지식 없이도 누구나 사용할 수 있습니다.

---

## 두 가지 사용 방법

| 방법 | 커맨드 | 언제 쓰나요? |
|------|--------|------------|
| **단건 생성** | `/소재생성` | 1개씩 빠르게 만들 때 |
| **대량 생성** | `/소재생성2` | 여러 개를 한 번에 만들 때 |

---

## 방법 1 — 단건 생성 (`/소재생성`)

### 순서

1. 슬랙 채널에서 `/소재생성` 입력
2. 팝업 모달에 내용 입력
3. **생성하기** 클릭
4. 잠시 후 스레드에 PNG 소재 자동 업로드

### 입력 항목

| 항목 | 필수 | 설명 |
|------|------|------|
| 템플릿 | ✅ | 아래 템플릿 목록 참고 |
| 메인 카피 | ✅ | 굵은 글씨 (48pt Bold) |
| 서브 카피 | ✅ | 작은 글씨 (39pt Regular) |
| 뱃지 텍스트 | ❌ | 코너 또는 인라인 배지 (템플릿에 따라 다름) |
| 상품 이미지 URL | ❌ | 비우면 채널 직전 업로드 이미지 자동 사용 |
| 좌측 메인/서브 카피 | ❌ | **비즈보드 전용** (좌측 텍스트 따로 입력) |

> 💡 **이미지 TIP**: 투명 배경 PNG(누끼 이미지)를 쓰면 자동으로 오브젝트형으로 합성됩니다.  
> 일반 JPG를 쓰면 rembg로 자동 누끼 처리 후 합성합니다.

---

## 방법 2 — 대량 생성 (`/소재생성2`)

### 구글 시트 바로가기

👉 **[소재 제작 시트 열기](https://docs.google.com/spreadsheets/d/1S536vDkW8Uy703TsxIrtZNYUNIFr114KqOFDr93-23M/edit?usp=sharing)**

### 순서

1. 위 시트에 소재 정보 입력 (아래 컬럼 구조 참고)
2. **F열(status)** 을 `제작요청` 으로 설정
3. 슬랙에서 `/소재생성2` 입력
4. 확인 팝업에서 건수 확인 후 **생성 시작** 클릭
5. 스레드에 소재 PNG가 행 순서대로 업로드됨
6. 완료된 행은 자동으로 `제작완료`로 변경됨

### 시트 컬럼 구조

| 열 | 컬럼명 | 필수 | 예시 |
|----|--------|------|------|
| A | `template` | ✅ | `기본_2줄형_좌측 오브제` |
| B | `main_copy` | ✅ | `레드윙 UP TO 13%` |
| C | `sub_copy` | ✅ | `베스트템 재입고` |
| D | `badge` | ❌ | `SALE` |
| E | `object_url` | ❌ | 누끼 PNG 이미지 URL |
| F | `status` | ✅ | `제작요청` → 봇이 처리 |
| G | `result_note` | 봇 자동 기입 | 완료 메모 |
| H | `main_copy_l` | ❌ | 비즈보드 좌측 메인 카피 |
| I | `sub_copy_l` | ❌ | 비즈보드 좌측 서브 카피 |

### F열 상태값 의미

| 값 | 의미 |
|----|------|
| `제작요청` | 봇이 처리할 대상 |
| `제작완료` | 처리 완료 (재실행 시 건너뜀) |
| `실패` | 처리 실패 — `제작요청`으로 바꾸면 재시도 가능 |

> ⚠️ **주의**: `/소재생성2`는 자동 누끼 처리가 없습니다. 오브젝트 이미지는 **투명 PNG(누끼 완료본)** 를 사용해야 합니다.

## 템플릿 종류

| 템플릿명 | 구성 | 오브젝트 이미지 | 뱃지 |
|---------|------|----------------|------|
| `비즈보드` | 좌카피 + 중앙 오브젝트 + 우카피 | ✅ 중앙 | ❌ |
| `기본_2줄형` | 좌카피 + 우측 오브젝트 | ✅ 우측 | ✅ 코너 |
| `기본_2줄형_좌측 오브제` | 좌측 오브젝트 + 우카피 | ✅ 좌측 | ✅ 코너 |
| `기본_2줄형_좌측 오브제+뱃지` | 좌측 오브젝트 + 우카피 + 로고 | ✅ 좌측 | ✅ 코너 |
| `썸네일형` | 카피 + 우측 썸네일 박스 | ✅ 우측 | ❌ |
| `앱다운로드형` | 앱 바 + 카피 | ❌ | ❌ |
| `앱다운로드+썸네일형` | 앱 바 + 카피 + 썸네일 | ✅ 우측 | ❌ |
| `텍스트강조+썸네일형` | 카피 + 오렌지 pill 배지 + 썸네일 | ✅ 우측 | ✅ pill |
| `텍스트강조형` | 카피 + 오렌지 pill 배지 | ❌ | ✅ pill |
| `텍스트강조+썸네일형 v2` | 카피 + 오렌지 텍스트 프리픽스 + 썸네일 | ✅ 우측 | ✅ 텍스트 |
| `텍스트강조형 v2` | 카피 + 오렌지 텍스트 프리픽스 | ❌ | ✅ 텍스트 |

## 카피 작성 주의사항 (카카오 검수 기준)

카카오 광고 검수에서 보류되지 않으려면 아래 가이드를 지켜주세요.

| 항목 | 기준 |
|------|------|
| 메인 카피 길이 | 최대 한 줄, 약 15자 이하 권장 |
| 서브 카피 길이 | 최대 한 줄, 약 20자 이하 권장 |
| 메인·서브 카피 | **다른 내용**이어야 함 (동일 문구 불가) |
| 금지 문자 | 이모티콘, 특수기호 (^^, ☞, ♨, 😀 등) |
| 줄바꿈 | 불가 (한 줄만 허용) |

## 오브젝트 이미지 준비 방법

### 투명 PNG(누끼) 이미지가 필요한 경우

- **`/소재생성` (v1 단건)**: 일반 이미지를 넣어도 자동으로 배경 제거 후 합성
- **`/소재생성2` (v2 대량)**: 자동 누끼 없음 → 미리 투명 PNG로 준비 필요

### 추천 누끼 처리 방법

| 방법 | 특징 |
|------|------|
| [remove.bg](https://www.remove.bg/) | 무료 (월 50장), 품질 좋음 |
| [Clipping Magic](https://clippingmagic.com/) | 유료, 세밀한 수동 편집 가능 |
| Photoshop 피사체 선택 | 자체 제작 가능 시 |

## 자주 묻는 질문

**Q. 소재가 스레드에 안 올라와요**  
→ 봇이 처리 중일 수 있습니다. 약 10~30초 기다려 주세요.  
→ 이미지 URL이 잘못됐거나 외부에서 접근 불가한 URL일 수 있습니다.

**Q. "제작요청"으로 바꿨는데 봇이 안 돌아요**  
→ `/소재생성2`를 직접 입력해야 봇이 시트를 읽습니다. 자동으로 실행되지 않습니다.

**Q. 실패한 행을 다시 처리하고 싶어요**  
→ F열 상태값을 `제작요청`으로 변경 후 `/소재생성2` 다시 실행하면 됩니다.

**Q. 비즈보드에서 좌측·우측 카피를 따로 쓰려면?**  
→ 시트의 H열(main_copy_l), I열(sub_copy_l)에 좌측 카피를 별도 입력하세요.  
→ 비워두면 B열(main_copy), C열(sub_copy)로 좌우 동일하게 들어갑니다.

**Q. 소재 크기가 맞지 않아요**  
→ 봇은 항상 **1029×258px PNG**를 출력합니다. 카카오 비즈보드 규격에 맞습니다.
